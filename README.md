## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/rick-morty.git
cd rick-morty
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open your browser and visit `http://localhost:5173`
